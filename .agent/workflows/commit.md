# /commit Workflow

<objective>
Commit changes to the repository adhering to the project's commit message conventions.
</objective>

<process>
1. **Analyze Changes**: Check `git status` and `git diff` to understand what is being committed.
2. **Execute Git Committer Skill**: Use the `Git Committer` skill (.agent/skills/git-committer/SKILL.md) to draft and execute the commit.
</process>
